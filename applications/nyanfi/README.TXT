●動作環境
　Windows Vista 〜 Windows 10 で多分大丈夫です。
　こちらでは Windows 8.1 および Windows 10 で動作確認を行っています。

　V10.52 から Windows XP 非対応となっています。
　XP でも動作する最終版は公開サイトにありますので、必要な方はダウンロードしてください。 

●インストール
　適当なディレクトリに以下のファイルを置いてください。

　NyanFi.exe NyanFi本体 
　NyanFi.chm ヘルプファイル 

　ZIP、7z、LHA、CAB、TAR、RAR、ISO を扱うには、別途 7-zip32.dll、unlha32.dll、cab32.dll、tar32.dll、unrar32.dll、unrar.dll、uniso32.dll が必用です(RAR、ISO は展開のみ)。

　C/Migemo に対応しています。利用する場合は、migemo.dll (32bit版)や辞書ファイルをインストールしてください。

　xd2txlib.dll (xdoc2txt のDLL版)に対応しています。各種バイナリ文書に対して、テキストビュアー表示、文字列検索(GREP)などを行えます。

　なお、ファイルメニュー - [更新の確認]で、最新版のチェックおよび更新が可能です。

　source ディレクトリにはソースコード(C++Builder 10.2.3 用)がありますので、自由にご利用ください。 


●アンインストール
　インストールしたファイルおよび INIファイル等を削除して下さい。
　レジストリは使用していません。
